/*--------------------------------------------Program Files-------------------------------------------*/

  
   Server.c
   
   -   The purpose of this file is to initialize the connection with the client and provide remote
       Procedure call such as to list the files, change the current directory, reply do nothing, acknowledge
       user on quit
 

   Client.c
   
   -   The purpose of this file is to request the server to connect and send the commands such as list, cd,
       do nothing, quit and get the service from the server


/*-------------------------------Steps to set up and execute the program------------------------------*/


Step 1 :   type 'make' in the command line   / To compile the programs /
 
Step 2 :   type './Server' or './Client'  / To execute the single program /
