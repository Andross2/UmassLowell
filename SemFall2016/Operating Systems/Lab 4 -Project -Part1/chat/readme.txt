/*--------------------------------------------Program Files-------------------------------------------*/

  
   Server.c
   
   -   The purpose of this file is to intialize the connection to different clients who want connect chat
	room and send the messages betweebn the clients
 

   Client.c
   
   -   The purpose of this file is to request the server to connect and send the messages to the server
	so that they are forwarded to other clients.


/*-------------------------------Steps to set up and execute the program------------------------------*/


Step 1 :   type 'make' in the command line   / To compile the programs /
 
Step 2 :   type './Server' or './Client'  / To execute the single program /
